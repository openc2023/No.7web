## 修复菜单加号按钮显示问题

### 问题分析
在 `generateTreeHTML` 函数中，加号按钮的显示逻辑存在问题。当前代码只有当节点有子节点（`hasChildren` 为 true）时，才显示加号按钮，而实际上所有节点都应该可以添加子节点。

### 解决方案
1. 修改 `generateTreeHTML` 函数中的加号按钮显示逻辑，让所有节点都显示加号按钮
2. 确保 `addSubmenu` 函数能正确处理没有子节点的情况
3. 测试修复后的功能

### 具体修改
```javascript
// 原代码
${hasChildren ? 
    `<div class="action-btn" onclick="addSubmenu(event, '${item.id}')" title="New Page"><i class="fa-solid fa-plus"></i></div>` : 
    ''
}

// 修改为
<div class="action-btn" onclick="addSubmenu(event, '${item.id}')" title="New Page"><i class="fa-solid fa-plus"></i></div>
```

### 预期效果
- 所有节点都显示加号按钮
- 点击加号按钮可以添加子节点
- 子节点添加后，父节点自动展开
- 新节点默认显示为文件类型
